# COVID-19 Global Data Analysis

## Overview
End-to-end data analysis project on 589,779 rows of global COVID-19 
data covering 262 countries from January 2020 to May 2026.

## Tools Used
- SQL (SQLite / DB Browser for SQLite)
- Power BI (DAX measures, 4-page interactive dashboard)

## Dataset
Source: Our World in Data
https://ourworldindata.org/covid-deaths
Not included due to file size — download from link above.

## Project Files
- COVID19_Analysis.pdf — 4-page Power BI dashboard
- COVID_Project_Brief.docx — project brief and analysis plan
- COVID_Analysis_Report.docx — full findings report with charts

## Dashboard Pages
1. Cases & Deaths Overview
2. Vaccination Progress
3. Risk & Demographics Analysis
4. Deaths: Volume vs Severity

## Key Findings
- 779M total cases | 7.12M deaths | 0.91% global death rate
- US had most deaths (1.24M) — Peru had highest death rate (4.88%)
- 13.72 billion vaccine doses administered globally
- 65% global vaccination rate
- Europe hit hardest by total deaths (2.11M)
- Timing of lockdowns mattered more than strictness

## SQL Concepts Used
Data cleaning, CTEs, Window Functions, ROW_NUMBER(),
Temp Tables, JOINs, Subqueries, CASE WHEN

## Power BI / DAX Concepts Used
SUMX, FILTER, CALCULATE, MAX, DIVIDE,
AVERAGEX, DISTINCTCOUNT, TOPN, SELECTEDVALUE
